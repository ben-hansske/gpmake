# gpmake
a small generic build system using lua as an scriptable interface

## Requirements
+ Lua 5.3
+ cmake 3.12
+ C++17 compiler

### Ubuntu / Debian
```
sudo apt install cmake liblua5.3-dev
```
## Building it
```
git clone https://github.com/ben-hansske/gpmake
cd gpmake
mkdir build && cd build
cmake ..
make
```
